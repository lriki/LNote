 ■ LightNote ver1.00
===============================================================================

-----------------------------------------------------------------------
 ◆ 概要

    LightNote はゲーム制作に必要な機能をまとめた開発支援ライブラリです。
    現在は音声機能のみを公開しています。

    インストール方法や著作権情報は、同梱のマニュアルをご覧ください。


-----------------------------------------------------------------------
 ◆ パッケージ構成

    LNote_HSP
    ├ Doc             … マニュアルの入っているフォルダです
    ├ Sample          … サンプルの入っているフォルダです
    ├ lnote.as        … HSP 用のヘッダファイルです
    ├ LNote.dll       … プラグイン本体です
    ├ lnote.hs        … HSP 用のヘルプファイルです
    └ Manual.html     … マニュアルへのショートカットです
